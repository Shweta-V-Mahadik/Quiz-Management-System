# Quiz-Management-System
Quiz Management System developed during my internship at Sunbeam CDAC, Pune using Java, JDBC, JSP, and MySQL. The system supports admin quiz management, student participation, and database-driven quiz operations with a structured MV approach.
Quiz Management System

This project was developed as part of my internship at Sunbeam CDAC, Pune, where we designed and implemented a complete Quiz Management System using Java-based web technologies.

The application allows administrators to create, manage, and organize quizzes, while students can attempt quizzes and view results. The system follows a structured flow using JSP pages, JavaBeans, JDBC, and MySQL, ensuring proper separation of concerns and efficient database handling.

🛠️ Technologies Used

Java

JSP & Servlets

JDBC

MySQL

HTML & CSS

Apache Tomcat

✨ Key Features

Admin authentication and dashboard

Quiz creation and management

Question handling through database

Student quiz attempts

Result evaluation and storage

Secure and structured JDBC-based database interaction
